# DRAFT Requirements: v2.5 UI/UX Real-App Overhaul

**Status:** DRAFT — chờ user review
**Milestone:** v2.5 UI/UX Real-App Overhaul
**Created:** 2026-06-04
**Reference:** `.planning/research/v2.5-ui-ux-references.md`

> **Goal:** Nâng cấp UI/UX toàn app Peerlight AI cho 4 vai trò (Student/Teacher/Parent/Admin) theo chuẩn các app thực tế (Calm, Linear, Apple Health, Pi, Notion, Vercel...). Giữ brand violet/cyan, Vietnamese-first, privacy-first, no-surveillance.

> **Constraint cứng:** Không thay đổi backend API contract. Không thay đổi privacy boundaries. Không thêm tracking/analytics raw. Không break existing tests.

## UX-DS: Design System Foundation

| ID       | Requirement                                                                                                                        | Priority |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------- | -------- |
| UX-DS-01 | Mở rộng `ui-primitives.tsx` với Sheet, Command, Tabs, Tooltip, Progress, Dialog (Radix-based shadcn/ui pattern), presentation-only | Must     |
| UX-DS-02 | Typography scale chuẩn hoá (Be Vietnam Pro heading + Inter body), token-driven trong `tailwind.config.ts`                          | Must     |
| UX-DS-03 | Motion tokens chuẩn (fade-slide page transition, spring sheet, shimmer skeleton), respect `prefers-reduced-motion`                 | Should   |
| UX-DS-04 | Dark mode parity full app (Linear/Vercel standard), kiểm tra contrast WCAG AA                                                      | Must     |

## Constraints

- Không thay đổi backend API contract.
- Không thay đổi privacy boundaries (SOS-only adult visibility, reason-gate, metadata-only ops).
- Không thêm raw analytics, tracking pixels, third-party scripts.
- Không break smoke:demo, smoke:pilot, release-gates.
- Giữ Peerlight brand (violet `#7457e8` + cyan accent + Vietnamese-first).
- No browser token storage (cookie session unchanged).
- All existing tests (backend pytest, frontend vitest, lint, build) must remain green.
