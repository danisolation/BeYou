# Plan 06-02: Admin Reports Frontend UI

## Objective

Expose the backend aggregate report through a Vietnamese privacy-respecting admin UI.

## Tasks

1. RED: add Vitest coverage for API helper path, dashboard card, report rendering, scope filter, suppression labels, no token storage, and no raw/export/drilldown copy.

- Reports page renders required privacy notes and no raw export/per-student drilldown.
- Demo scope filter calls backend with the selected `demo_scope`.
- Suppressed buckets render `Đã ẩn để bảo vệ riêng tư (<3)`.
- UI uses existing `apiFetch` with cookies and no token storage.
