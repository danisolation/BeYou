# BeYou UI Feedback Specification (v1.7)

Source: User feedback with Google Stitch mockups
Date: 2026-05-27

## Key Principles
- UI must be synchronized across all pages (Stitch generates inconsistent designs)
- Most homepage elements are image+text blocks, NOT links to other pages
- History/logs belong INSIDE their respective feature pages, not on dashboard
- Platform prioritizes privacy - no auto-SOS, user controls sharing
- AI chatbot named "Peerlight AI" (not "AI Assistant")

## Student Pages

- Make cards (not links to other pages)
- Keep content as image+text blocks only
- "Bắt đầu" is the only link that navigates

### 2. Login
